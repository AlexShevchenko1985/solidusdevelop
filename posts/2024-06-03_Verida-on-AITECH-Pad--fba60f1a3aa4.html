Introduction

Verida is developing the world’s first private self-sovereign data DePIN for Web3.

The Verida Network provides secure and private database storage for users to own and control their private data.

Verida has developed the first private self-sovereign decentralized database storage network and has built a leading mobile wallet that supports zk proofs, crypto, identity, data storage and encrypted messaging.

Key Points

🔐 Secure, encrypted, private database storage for users to own and control their private data.
🔋 The Verida Storage Credit Token ($VDA) the native utility token, facilitating secure data storage and economic protocol incentivisation on the Verida Network.
🔑 Verida Wallet is the world’s first self-custody mobile application that brings data, Zero-Knowledge and crypto wallet into one mobile app.

🌎 Market: Database storage will be a $200b market by 2027.
